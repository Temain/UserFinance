# UserFinance
Микросервисная платформа на .NET 8 с Clean Architecture и CQRS для управления пользователями и получения персонализированных курсов валют с автоматическим обновлением данных из внешнего источника.

### Docker Compose
1. Перейдите в папку решения:
   `cd UserFinance`
2. При необходимости создайте локальный `.env`:
   `cp .env.example .env`
3. Запустите контейнеры:
   `docker compose up --build`

### Local Run With Global Environment Variables
Если удобнее один раз настроить переменные глобально, можно добавить их в `~/.zshrc` или `~/.zprofile`

После этого новые shell-сессии и IDE, запущенные из них, смогут использовать эти значения без настройки каждого проекта отдельно.

### Local Environment Script
Если не хочется прописывать переменные в `~/.zshrc` или `~/.zprofile`, можно один раз загрузить их в текущую shell-сессию из .envrc
